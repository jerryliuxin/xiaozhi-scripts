# 0000 — System Setup Template

## Date
2026-06-17

## Purpose
This is a template for learning records. Copy this file and rename to `NNNN-topic.md` when starting a new learning session.

## Template Fields
- **Date**: When the lesson occurred
- **Content Covered**: What was taught
- **Mickey's Performance**: Right/wrong, strengths/weaknesses
- **Key Insights**: Non-obvious findings worth remembering
- **Next Steps**: What to review or introduce next

## Usage
1. Copy this file
2. Rename to `NNNN-<topic>.md` (increment NNNN)
3. Fill in the fields after each lesson
4. The insights here will inform future lessons
