# 小学数学公式速查

记录米奇需要掌握的小学5-6年级数学概念。

## 数的分类
- 质数 (prime number): 只有 1 和自身两个因数
- 合数 (composite number): 除了 1 和自身还有其他因数
- 奇数 (odd number): 不能被 2 整除
- 偶数 (even number): 能被 2 整除

## 因数与倍数
- 最大公因数 (GCD)
- 最小公倍数 (LCM)
- 因数分解

## 分数
- 约分 (simplify)
- 分数转百分数
- 分数加减乘除

## 几何公式
- 长方形面积: `A = l × w`
- 长方形周长: `P = 2(l + w)`
- 三角形面积: `A = ½ × b × h`
- 梯形面积: `A = ½ × (a + b) × h`
- 圆面积: `A = πr²`
- 圆周长: `C = 2πr`

## 运算定律
- 交换律 (commutative): `a + b = b + a`, `a × b = b × a`
- 结合律 (associative): `(a + b) + c = a + (b + c)`
- 分配律 (distributive): `a × (b + c) = a×b + a×c`

## 小数
- 小数转百分数
- 小数加减乘除
